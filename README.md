# qa_python

Финальный проект спринта 4.  
Юнит-тестирование класса `BooksCollector`.

Класс `BooksCollector` хранит книги, их жанры и список избранного.  
Цель проекта — покрыть основные методы класса юнит-тестами на pytest.

Реализованные тесты:

- добавление новой книги в словарь `books_genre`;
- параметризованный тест на невалидные названия книг (пустое имя и имя длиннее 40 символов);
- установка корректного жанра книге;
- проверка, что некорректный жанр не устанавливается;
- получение жанра книги по названию;
- получение списка книг с конкретным жанром;
- получение полного словаря `books_genre`;
- получение списка книг, подходящих детям (исключены жанры из `genre_age_rating`);
- добавление книги в избранное без дублей;
- удаление книги из избранного;
- получение актуального списка избранных книг.

В проекте используются:

- `pytest`;
- фикстура `collector`, создающая экземпляр `BooksCollector` для тестов;
- параметризация теста `test_add_new_book_invalid_name_not_added`.
